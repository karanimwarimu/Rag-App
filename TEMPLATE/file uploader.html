Uploader Manual

  • 1. Select Files

    Click the "browse" link or drag and drop one or more files into the designated area.

    Accepted formats: .jpg, .jpeg, .png, .docx, .txt, .pdf.

    Note: Images should contain text for best results.

  • 2. View Queue

    The files will appear in a list below, ready for sequential processing. You can track progress and manage files before they upload.

  • 3. Monitor Progress

    Each file will be uploaded one after another. You will see a green checkmark and a toast notification upon successful completion.

  • 4. Access Assistant

    Click the circular button in the bottom-right corner to access the virtual assistant for help or guidance.
